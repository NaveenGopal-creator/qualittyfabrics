WhatsApp Backend Integration

A complete developer guide — from simple wa.me links to full WhatsApp Business API with Node.js backend and email notifications.

Currently active on this website: The contact form sends enquiry details directly to WhatsApp via wa.me link. To receive form data by email or a CRM, follow the steps below.

Choose Your Integration Method
There are 3 common approaches to WhatsApp backend integration. Pick based on your budget and technical needs.

Method A — wa.me Link

Form data is formatted and user clicks to open WhatsApp with pre-filled message. No server needed.

✅ Free  ✅ No server  ✅ Already implemented
❌ User must click  ❌ No email backup

Method B — Email + WA Link

Form submits to a backend (Node.js/PHP). Backend emails you AND opens WA link for user. Best for small business.

✅ You get email copy  ✅ Cheap (free tier)
❌ Needs basic server  ❌ User still clicks WA

Method C — WA Business API

Full official WhatsApp Business API. Server auto-sends WA message to your number without user interaction.

✅ Fully automated  ✅ Professional
❌ Paid (~₹5–15/msg)  ❌ Needs Meta approval

Method D — Formspree / Netlify

No-code solution. Point your form to Formspree — they email you on every submission. Free tier available.

✅ Zero code  ✅ Free tier (50 submissions/mo)
❌ No WA auto-send  ❌ Branding on free tier

Method B Node.js Backend — Email + WhatsApp
This is the recommended approach. Set up a simple Node.js server that receives the form, emails you the enquiry, and optionally sends a WhatsApp message via Twilio.

Step 1 — Install Node.js packages

npm init -y
npm install express nodemailer cors dotenv

Step 2 — Create server.js

// server.js — Qualitty Fabrics Contact Form Backend
require('dotenv').config();
const express  = require('express');
const nodemailer = require('nodemailer');
const cors     = require('cors');
const app      = express();

app.use(cors({ origin: 'https://www.qualittyfabrics.in' }));
app.use(express.json());

// Email transporter (using Gmail — use App Password)
const transporter = nodemailer.createTransport({
  service: 'gmail',
  auth: {
    user: process.env.EMAIL_USER,   // your Gmail
    pass: process.env.EMAIL_PASS,   // Gmail App Password
  }
});

app.post('/submit-enquiry', async (req, res) => {
  const { name, company, email, phone, fabric, qty, country, message } = req.body;

  // Validate required fields
  if (!name || !email || !fabric) {
    return res.status(400).json({ error: 'Missing required fields' });
  }

  // Build email content
  const emailBody = `
    New Enquiry — Qualitty Fabrics Website
    =======================================
    Name     : ${name}
    Company  : ${company || '—'}
    Email    : ${email}
    Phone    : ${phone || '—'}
    Country  : ${country || '—'}
    Fabric   : ${fabric}
    Quantity : ${qty || '—'}
    Message  : ${message || '—'}
    =======================================
    Received : ${new Date().toLocaleString('en-IN', { timeZone: 'Asia/Kolkata' })}
  `;

  try {
    // 1. Send email to Qualitty Fabrics team
    await transporter.sendMail({
      from: `"Qualitty Fabrics Website" <${process.env.EMAIL_USER}>`,
      to: 'info@qualittyfabrics.in',       // <-- your email here
      replyTo: email,
      subject: `New Fabric Enquiry: ${fabric} — ${name}`,
      text: emailBody,
    });

    // 2. Send confirmation to the customer
    await transporter.sendMail({
      from: `"Qualitty Fabrics" <${process.env.EMAIL_USER}>`,
      to: email,
      subject: 'We received your enquiry — Qualitty Fabrics',
      text: `Dear ${name},\n\nThank you for your enquiry about ${fabric}.\nOur team will respond within 24 hours.\n\nBest regards,\nQualitty Fabrics\n+91 97159 88111\nwww.qualittyfabrics.in`,
    });

    res.json({ success: true });
  } catch (err) {
    console.error(err);
    res.status(500).json({ error: 'Email failed' });
  }
});

app.listen(3000, () => console.log('Server running on port 3000'));

Step 3 — Create .env file

# .env — DO NOT commit this to GitHub
EMAIL_USER=yourname@gmail.com
EMAIL_PASS=your_gmail_app_password
Get Gmail App Password: Google Account → Security → 2FA → App Passwords → Generate. Use that 16-char password, NOT your Gmail password.

Step 4 — Update contact.html form submit function

async function submitForm() {
  const name   = document.getElementById('f_name').value.trim();
  const email  = document.getElementById('f_email').value.trim();
  const fabric = document.getElementById('f_fabric').value;
  if (!name || !email || !fabric) { alert('Please fill required fields.'); return; }

  const payload = {
    name,
    email,
    company  : document.getElementById('f_company').value,
    phone    : document.getElementById('f_phone').value,
    country  : document.getElementById('f_country').value,
    fabric,
    qty      : document.getElementById('f_qty').value,
    message  : document.getElementById('f_msg').value,
  };

  try {
    const res = await fetch('https://YOUR-SERVER-URL/submit-enquiry', {
      method : 'POST',
      headers: { 'Content-Type': 'application/json' },
      body   : JSON.stringify(payload),
    });
    const data = await res.json();
    if (data.success) {
      document.getElementById('formBody').style.display = 'none';
      document.getElementById('formSuccess').style.display = 'block';
      // Also open WA so customer can follow up instantly
      const waText = `New enquiry from ${name} (${email}) for ${fabric}`;
      window.open('https://wa.me/919715988111?text=' + encodeURIComponent(waText), '_blank');
    }
  } catch (err) {
    alert('Something went wrong. Please try again or WhatsApp us directly.');
  }
}

Method C WhatsApp Business API (Official)
This sends an automated WhatsApp message to your business number when someone fills the form — fully hands-free. Requires a Meta Business account and approval (takes 1–3 days).

Setup (one-time)

1. Create Meta Business account at business.facebook.com  |  2. Add WhatsApp Product  |  3. Get a permanent Access Token  |  4. Note your Phone Number ID

Add to server.js — after sending email, add this:

// WhatsApp Business API — send message to your own number
const WA_TOKEN   = process.env.WA_ACCESS_TOKEN;   // from Meta
const WA_PHONE_ID = process.env.WA_PHONE_NUMBER_ID; // from Meta

async function sendWhatsAppAlert(data) {
  const { name, fabric, email, phone } = data;
  const url = `https://graph.facebook.com/v18.0/${WA_PHONE_ID}/messages`;

  await fetch(url, {
    method : 'POST',
    headers: {
      'Authorization': `Bearer ${WA_TOKEN}`,
      'Content-Type' : 'application/json',
    },
    body: JSON.stringify({
      messaging_product: 'whatsapp',
      to  : '919715988111',      // YOUR WhatsApp number with country code
      type: 'text',
      text: {
        body: `🧵 *New Enquiry — Qualitty Fabrics*\n\n*Name:* ${name}\n*Fabric:* ${fabric}\n*Email:* ${email}\n*Phone:* ${phone || '—'}\n\nReply to this message to respond.`
      }
    })
  });
}

Method D No-Code Option — Formspree (Easiest)
If you don't want to manage a server at all, Formspree handles form submissions and emails them to you. Free for 50 submissions/month — paid plans for more.

Step 1 — Sign up and get your form endpoint

Go to formspree.io → New Form → Copy your form ID (looks like: xaybzzzz)

Step 2 — Replace submitForm() in contact.html

async function submitForm() {
  const name   = document.getElementById('f_name').value.trim();
  const email  = document.getElementById('f_email').value.trim();
  const fabric = document.getElementById('f_fabric').value;
  if (!name || !email || !fabric) { alert('Please fill required fields.'); return; }

  const formData = new FormData();
  formData.append('name',    name);
  formData.append('email',   email);
  formData.append('company', document.getElementById('f_company').value);
  formData.append('phone',   document.getElementById('f_phone').value);
  formData.append('fabric',  fabric);
  formData.append('qty',     document.getElementById('f_qty').value);
  formData.append('country', document.getElementById('f_country').value);
  formData.append('message', document.getElementById('f_msg').value);

  try {
    const res = await fetch('https://formspree.io/f/YOUR_FORM_ID', {
      method : 'POST',
      body   : formData,
      headers: { 'Accept': 'application/json' }
    });
    if (res.ok) {
      document.getElementById('formBody').style.display = 'none';
      document.getElementById('formSuccess').style.display = 'block';
    }
  } catch (err) {
    alert('Submission failed. Please WhatsApp us directly.');
  }
}
That's it! Formspree will email info@qualittyfabrics.in for every submission. No server needed.

Deploy the Node.js Backend (Free)
Best free hosting options for your Node.js server:

Railway.app

Easiest. Connect GitHub repo → auto deploy. Free $5/month credit. Recommended for beginners.

✅ railway.app

Render.com

Free tier available. Slightly slower cold starts but reliable. Good for production.

✅ render.com

AWS / DigitalOcean

Most professional. AWS EC2 free tier or DigitalOcean Droplet (~₹400/month). Best for scale.

✅ aws.amazon.com / digitalocean.com

Vercel (Serverless)

Convert server.js to a serverless function. Vercel hosts for free. No server management.

✅ vercel.com — Free tier
